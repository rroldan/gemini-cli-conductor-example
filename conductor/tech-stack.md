# Tech Stack

## Language
- **Rust**: A systems programming language that provides memory safety without garbage collection, making it ideal for high-performance and reliable backends.

## Web Framework
- **Axum**: An ergonomic, modular web framework built on top of `tower`, `hyper`, and `tokio`, providing a clean and intuitive API for building web services in Rust.

## Database Interaction
- **SQLx**: An async-native, pure Rust SQL toolkit with compile-time checked queries, ensuring type safety and performance for PostgreSQL interactions.

## Database System
- **PostgreSQL**: A powerful, open-source object-relational database system known for its reliability, feature robustness, and performance.
